# bill-splitter-web
Simple bill splitting web app built using HTML , CSS and JS

A simple and responsive web application built with **HTML, CSS, and JavaScript** that helps users split bills among friends or group members.  

## ✨ Features
- Add total bill amount and number of people
- Calculate each person’s share instantly
- Option to include tips or extra charges
- Responsive design for mobile and desktop
- Lightweight and easy to use (no frameworks required)

## 🚀 Demo
Open `index.html` in your browser to run the app locally.  
You can also host it on GitHub Pages, Netlify, or Vercel for free.
